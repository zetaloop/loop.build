# 欢迎使用 MkDocs

访问 [mkdocs.org](https://www.mkdocs.org) 来阅读完整文档。

## 命令

* `mkdocs new [目录名]` - 新建项目.
* `mkdocs serve` - 启动自动刷新的调试服务器.
* `mkdocs build` - 构建文档站点.
* `mkdocs -h` - 打印帮助信息并退出.

## 项目布局

    mkdocs.yml    # 配置文件.
    docs/
        index.md  # 文档主页.
        ...       # 其他页面、图像、文件等.
