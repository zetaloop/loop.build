# 我才知道 Markdown 行尾加两个空格可以换行

刚刚问 GPT 才知道的。

不过这个语法我觉得多少有点令人不适，太难看了。反正我是没法接受任何形式的行尾空格，编辑器里全部自动标红。<br>
我马上去把 MkDocs 的软换行打开，这样就不用手动 `<br>` 了。
<!-- more -->

```yaml title="mkdocs.yml"
markdown_extensions:
  - markdown.extensions.nl2br
```
